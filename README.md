## 心理建设

为了保证复试的成功，需要不留余地地进行复习！因此，任务就会较为严峻，但仍需以一种俾睨天下的态度对待此次准备。最难的初始都已经熬过来了，难道还会怕这小小复试吗？——不足为惧！

## 项目准备

现如今正值AI时代，各种大模型层出不穷，并且各个行业都在充分地与大模型进行结合，更不用说在学校里搞研究的导师们了，在项目准备这一块推荐准备一个RAG项目和Agent项目，再来一个常规的前后端开发（如果是科班的话，一般学校的实训就会有这样的项目）。

要准备写这两个ai项目，就得了解现如今大模型的核心原理，以防老师问到不清楚！
